Git init: 
app.py ve requirements.txt dosyalarının bulunduğu klasörü, yeni bir Git Repository olarak başlatıyorum.

git remote add origin https://github.com/mert-kilicaslan/BCFM-23-Akademi.git: 
Komutu ile localimdeki Git Repositoryi bir GitHub’daki remote repositorye (https://github.com/mert-kilicaslan/BCFM-23-Akademi.git) bağlamak/ilişkilendirmek için kullandım.

Git add:
Bu komut ile app.py ve requirements.txt dosyalarını Git'e ekledim.

Git commit: 
Bu komut ile ‘git add’ komutunda eklemiş olduğum değişiklikleri kaydettim. m parametresi ile beraber yapılan/eklenen değişiklik hakkında ufak bir açıklama ekledim.

git push -u origin main: 
Komutu ile Local Git Repository’de yapmış olduğum değişiklikleri GitHub’daki repository’nin main branch’ine gönderdim.
